## ライセンス本文

[OFL.txt](OFL.txt) を参照。 全フォント共通の SIL Open Font License Version 1.1 本文を 1 つ同梱している。 各フォントの個別 OFL.txt はファミリ名と Copyright 行のみ差異があるが、 ライセンス条文 (Permission / Conditions / Termination 等) は共通。

## 利用上の注意

- 同梱したフォントは 「Reserved Font Name」 として元の名称を維持して配布している (改名 / 派生していない)
- フォント単体を販売することは禁止 (OFL §3 — 同梱したアプリケーションは販売可能)
- 同梱・再配布する際は OFL ライセンスファイルを併せて配布すること (OFL §2 — 本ディレクトリでは `OFL.txt` を同梱)
