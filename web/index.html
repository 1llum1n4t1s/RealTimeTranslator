Features

音だけ。あとは字幕で世界を渡る。

Just audio in. Subtitles out, in any language.

ゲームも動画も配信も、指定したプロセスの音だけを掴んでリアルタイム翻訳。賢く・安く・うっとうしくない設計です。 Capture one process's audio, translate it live, and stay out of your way — smart, cheap, and unobtrusive by design.

Per-process Audio Capture

プロセス単位での音声キャプチャ

ゲーム・動画プレーヤー・ブラウザなど、指定したアプリの音だけをキャプチャ。マイクや BGM を巻き込みません。 Pick a target process and capture only its audio — never your mic, never the BGM playing in another window.

WASAPILoopbackPer-app
Realtime API Streaming

Realtime API へのストリーミング翻訳

OpenAI Realtime API に音声を流し込み、聞こえた端から翻訳テキストを返してもらいます。 Streams audio straight into the OpenAI Realtime API and receives translated text as soon as the model hears it.

Click-through Overlay

クリック透過の最前面字幕

半透明オーバーレイで常に最前面に表示。マウス操作はそのまま下のアプリに通り抜けます。 Translucent always-on-top subtitle window — clicks pass straight through to the app underneath.

VAD Silence Skip

VAD で無音をスキップしコスト削減

音声区間検出（VAD）で無音をきれいに飛ばし、Realtime API への課金時間を最小化します。 Voice activity detection trims out silence, so you only pay the Realtime API for time someone is actually speaking.

Session Cost Meter

経過 / 推定 USD / 節約秒を可視化

セッション時間・推定課金額・VAD が節約した時間をリアルタイム表示。財布の見える化。 Live readout of elapsed time, estimated USD charge, and seconds saved by silence skipping. Your wallet, on screen.

ADV-style Translation Log

ADV 風の翻訳ログと保持期間設定

直近の発話を ADV ゲーム風のログで遡れます。保持期間や表示行数も自由に設定可能。 Scroll back through recent lines in a visual-novel-style log. Retention window and line count are both configurable.

Japanese Subtitle Fonts

字幕用日本語フォント 5 種を同梱

字幕の読みやすさにこだわった日本語フォントを 5 種同梱。気分やゲームの雰囲気に合わせて選べます。 Ships with five hand-picked Japanese fonts tuned for subtitle readability — switch them on a whim.
