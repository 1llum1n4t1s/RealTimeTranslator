How it works · 仕組み

音声をキャプチャし、訳して、字幕に。

Capture one app's audio, stream it to the OpenAI Realtime API, and paint the translation back as live subtitles — end to end, in real time.
STEP 01 →

プロセス音声をキャプチャPer-app audio capture

WASAPI Process Loopback で、選んだアプリ（ゲーム / 動画 / 配信クライアント）の音声だけを取り込みます。マイクもデスクトップ全体も巻き込みません。 WASAPI Process Loopback grabs only the chosen process's audio — not your mic, not the whole desktop.

STEP 02 →

Realtime API へストリーミングStream to OpenAI

PCM16 / 24kHz の音声を WebSocket で OpenAI Realtime Translate API（gpt-realtime-translate）へ送り、翻訳テキストを逐次受信します。 PCM16/24kHz audio streams over WebSocket to the Realtime Translate API, returning translated text token by token.

STEP 03

オーバーレイ字幕に表示Overlay subtitles

透過・最前面・クリック透過のオーバーレイ窓に字幕を描画。フォント・色・表示時間・行数は自由に調整できます。 Translations appear in a transparent, always-on-top, click-through overlay. Font, color, duration and line count are all configurable.

Features · 機能

字幕を、あなた好みに。コストも見える。

Built for long sessions: cut API cost with a VAD gate, style the captions, and keep a searchable translation log.

VAD ゲートで API コストを節約Silero VAD gate & live cost meter

Silero VAD（ONNX）が「人の声らしさ」を判定し、無音区間の送信を抑制。OpenAI Realtime API は送った音声をすべて課金するため、BGM の垂れ流しを止めるだけで大きく節約できます。プリセット（Balanced / 頭尻尾重視 / 節約重視 / Custom）を用意。経過時間・推定トークン・推定コスト・節約秒数をリアルタイム表示します。 Silero VAD suppresses silence so you don't pay to stream dead air. Choose a preset (Balanced / edge-priority / savings / custom) and watch a live meter of elapsed time, tokens, estimated USD cost and seconds saved.

BYOK · Bring Your Own Key

透明性について。Honest about your data & billing.

本アプリは BYOK（Bring Your Own Key） モデルです。あなた自身の OpenAI API キーを使い、キャプチャした音声はあなたの OpenAI アカウントの Realtime API に送信されます。課金もあなたのアカウントから発生します。完全なローカル処理ではありません。 RealTimeTranslator is BYOK. You supply your own OpenAI API key, audio is sent to YOUR OpenAI account's Realtime API, and billing happens on YOUR account. It is not fully local — we want to be clear about that.

アプリ音声 App audio → あなたの OpenAI アカウント Your OpenAI account → 翻訳字幕 Subtitles
  • API キーは Windows DPAPI で暗号化して保存（別 PC では復号不可）。 Your API key is encrypted with Windows DPAPI and can't be decrypted on another machine.
  • トラッキングや解析の埋め込みなし。中央サーバーは経由しません。 No analytics or tracking. Audio does not pass through any server of ours.
  • 音声は OpenAI に送信されます。OpenAI の利用規約・データ方針が適用されます。 Audio is sent to OpenAI; their terms and data policies apply to it.
